## Installation Steps

1. Navigate to frontend:
   ```bash
   cd frontend
   ```

2. Install core dependencies:
   ```bash
   npm install
   ```

3. Install additional packages:
   ```bash
   npm install axios react-router-dom
   ```

4. Create .env.local:
   ```bash
   copy .env.example .env.local
   ```
   Or manually create with:
   ```
   REACT_APP_API_URL=http://localhost:8000/api/v1
   ```

5. Start the app:
   ```bash
   npm start
   ```
   Or use the provided batch file:
   ```bash
   start.bat
   ```

## Environment Configuration

### .env.local Variables
- `REACT_APP_API_URL` - Backend API base URL (default: http://localhost:8000/api/v1)
- `REACT_APP_APP_NAME` - Application name (default: YouTube Clone)

## Authentication Flow

1. **Registration**
   - User provides: fullName, username, email, password, avatar, coverImage
   - Avatar is required, coverImage is optional
   - Files are uploaded as multipart/form-data
   - Backend uploads to Cloudinary
   - Access and refresh tokens returned

2. **Login**
   - User provides: username/email and password
   - Credentials validated
   - Tokens stored in localStorage
   - User redirected to dashboard

3. **Token Management**
   - Access token in Authorization header
   - Refresh token stored for renewal
   - Automatic refresh on 401 response
   - Logout clears all tokens

- `POST /users/logout` - Logout (requires auth)

## Troubleshooting

### Module not found errors
```bash
npm install
npm install axios react-router-dom
```

### CORS Issues
- Verify backend CORS config includes frontend URL
- Check console for exact error

### API Connection Failed
- Ensure backend runs on http://localhost:8000
- Check .env.local has correct API_URL
- Verify backend is responding

### Port 3000 Already in Use
```bash
# Windows: Kill the process
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Or use a different port
set PORT=3001 && npm start
```
